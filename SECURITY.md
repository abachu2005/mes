# Security policy

MES is **research software**, not a regulated medical device. Do not deploy it with protected health information (PHI) without appropriate IRB and security review.

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.2.x   | Yes       |
| < 0.2   | No        |

## Reporting a vulnerability

Email the maintainers via GitHub **Security advisories** on [abachu2005/mes](https://github.com/abachu2005/mes/security/advisories/new) or open a private security report.

Include steps to reproduce, impact, and suggested fix if any. We aim to acknowledge reports within 14 days.
